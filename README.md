# 🚀 Algorithm Master

📚 本项目记录我的算法学习过程，包含各种算法实现、解题思路和学习笔记。

## 📂 目录结构

- 🧩 `/sorting`: 排序算法实现
- 🔍 `/searching`: 搜索算法实现
- 🏗️ `/data_structures`: 数据结构实现

- 📝 `/notes`: 学习笔记

1. 📌 每个算法实现都包含详细的注释说明

4. 🧠 高级算法：动态规划、贪心算法、回溯算法
